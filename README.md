## 📝 Описание программы

Book Tracker - это приложение с графическим интерфейсом для учета прочитанных книг. Программа позволяет вести список книг, фильтровать их по различным критериям и сохранять данные в JSON файл.

### Основные возможности:

- ✅ Добавление книг с указанием названия, автора, жанра и количества страниц
- ✅ Просмотр списка всех книг в удобной таблице
- ✅ Фильтрация книг по жанру
- ✅ Фильтрация книг по количеству страниц (больше указанного значения)
- ✅ Удаление книг из списка
- ✅ Статистика по прочитанным книгам
- ✅ Автоматическое сохранение данных в JSON файл
- ✅ Валидация вводимых данных
