# Expense Tracker (Трекер расходов)

**Автор:** Михалкин Анатолий

## Описание
GUI-приложение для учёта личных расходов. Позволяет добавлять, удалять, фильтровать
расходы по категории и дате, а также подсчитывать сумму за выбранный период. Данные
сохраняются в формате JSON.

## Как использовать

### Запуск
1. Убедитесь, что установлен Python 3.7+.
2. Склонируйте репозиторий:
```

git clone <ссылка-на-репозиторий>
cd expense-tracker

```
3. Запустите приложение:
```

python main.py

```

### Интерфейс
- **Сумма** — положительное число (например, 150.50).
- **Категория** — выберите из списка.
- **Дата** — в формате ГГГГ-ММ-ДД (например, 2025-03-15).
- Нажмите **«Добавить расход»**.
- Для фильтрации укажите категорию и/или диапазон дат, нажмите **«Фильтровать»**.
- Для подсчёта суммы за период — введите даты и нажмите **«Подсчитать сумму»**.

### Запуск тестов
```

python -m unittest test_tracker.py

```

## Примеры использования

1. Введите: Сумма = 500, Категория = Еда, Дата = 2025-04-01 → нажмите «Добавить расход».
2. В фильтре: Категория = Еда, С даты = 2025-01-01, По дату = 2025-12-31 → «Подсчитать сумму».

## Требования
- Python 3.7+
- Стандартная библиотека tkinter
